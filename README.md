# fake.js

## install

```
npm i @sigodenjs/fake
```
## usage

```js
const fake = require('@sigodenjs/fake')
// require('@sigodenjs/lib/cn') providers `cnname` `cnsentence` ...
fake.integer()
fake.integer(1, 10)
```
